====test====
When I send a request to an operation
And in the path I specify ?__code=404
And the operation has 404 status code response defined
And in this operation there is no schema or example defined in the response
Then I get back a response with Content-Type of text/plain
====spec====
swagger: "2.0"
paths:
  /todos:
    get:
      responses:
        404:
          description: no resource
====server====
mock -p 4010
====command====
curl -i -X GET http://localhost:4010/todos?__code=404
====expect====
HTTP/1.1 404 Not Found
content-type: text/plain
Connection: keep-alive
