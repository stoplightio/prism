====test====
Requesting a default example results in sending back the first and only example for an operation.
====spec====
swagger: "2.0"
paths:
  /todos:
    get:
      produces:
        - application/json
      responses:
        200:
          description: Get Todo Items
          examples:
            application/json: "hello"
====server====
mock -p 4010
====command====
curl -i -X GET http://localhost:4010/todos?__example=default
====expect====
HTTP/1.1 200 OK
content-type: application/json; charset=utf-8

"hello"
