====test====
If I have an operation with a parameter with the allowEmptyValue set to false to false
and I send a request with a correcly encoded parameter and with the parameter empty
I should get a 200 response
====spec====
swagger: '2.0'
paths:
  /path:
    post:
      produces:
        - text/plain
      consumes:
        - application/x-www-form-urlencoded
      responses:
        200:
          schema:
            type: string
      parameters:
        - in: formData
          type: string
          name: id
          allowEmptyValue: true
====server====
mock -p 4010 ${document}
====command====
curl -i -X POST http://localhost:4010/path -H "Content-Type: application/x-www-form-urlencoded" --data "id"
====expect====
HTTP/1.1 200 OK

string
