====test====
Test sending form data with a missing property and invalid string.
Expect to get validation errors in return.
====spec====
openapi: '3.0.1'
paths:
  /path:
    post:
      responses:
        200:
          content:
            text/plain:
              example: ok
      requestBody:
        content:
          application/x-www-form-urlencoded:
            schema:
              type: object
              properties:
                id:
                  type: integer
                status:
                  type: string
                  enum:
                    - open
                    - close
              required:
                - id
                - status
====server====
mock -p 4010 ${document}
====command====
curl -i -X POST http://localhost:4010/path -H "Content-Type: application/x-www-form-urlencoded" --data-urlencode "status=ooopsie!"
